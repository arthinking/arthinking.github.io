#安装JDK

ubuntu 12.04 LTS 安装配置JDK

下载jdk-xxx.bin

解压文件
chmod u+x jdk-xxx.bin
./jdk-xxx.bin

复制解压出jdk-xxx文件夹到/usr/lib下
sudo mkdir -p /usr/lib/jvm/
cp -r jdk-xxx /usr/lib/jvm/jdkxxx

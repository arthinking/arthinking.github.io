## Mac终端配置

### 颜色配置

Mac OS X的终端默认是黑白的,下面是配置颜色的步骤:

### 1 设置常用命令语法高亮

修改`~/.bash_profile`文件,加入以下内容:

```
alias ls='ls -G'
alias ll='ls -l'
alias grep='grep --color'
```

这样,ls和grep命令就是彩色的了.

### 2 设置vim语法高亮

修改`~/.vimrc`文件,加入以下内容:

```
syntax on
```

保存退出后再次使用vim就会发现高亮语法了.
